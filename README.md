📱 Motorola Sales Insights Dashboard

# 📊 Project Overview
Interactive Power BI dashboard for Motorola Sales Analysis (41M Revenue). It provides actionable business insights for sales optimization by tracking monthly trends and regional performance.

# 🛠️ Technical Stack & Skills
- Tool: Power BI Desktop
- Data Cleaning: Power Query
- Analysis: DAX Measures for Revenue & KPI tracking
- Visuals: Map charts, Line trends, and Payment behavior analysis

# 📈 Key Insights
- Sales Trends: High volume identified during the year-end months.
- Geography: Regional sales distribution across major hubs.
- Payments: Analysis of customer preferences (UPI, Cash, etc.).
